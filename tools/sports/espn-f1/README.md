# espn-f1

Get Formula 1 race results, schedule, and driver standings from ESPN.

## Voice Triggers

- "Hey Cal, Who won the last F1 race?"
- "Hey Cal, What's the F1 schedule?"

## Required Services



## Setup

No environment variables required.

No credentials required.

## Installation

### Via CAAL Tools Panel (Recommended)

1. Open CAAL web interface
2. Click Tools panel (wrench icon)
3. Search for "espn-f1"
4. Click Install and follow prompts

### Via Command Line

```bash
curl -s https://raw.githubusercontent.com/CoreWorxLab/caal-tools/main/scripts/install.sh | bash -s espn-f1
```

## Usage

Formula 1 data suite - results, standings, and schedule. Parameters: action (required) - 'results', 'standings', or 'schedule'; driver (optional for standings) - driver name like 'Verstappen', 'Hamilton'. Examples: 'who won the last F1 race', 'F1 standings', 'when is the next grand prix'.
