# espn-get-nhl-schedule

Gets upcoming NHL games for a specified team with dates and opponents.

## Voice Triggers

- "Hey Cal, When do the Canucks play next?"
- "Hey Cal, What's the Maple Leafs schedule?"

## Required Services



## Setup

No environment variables required.

No credentials required.

## Installation

### Via CAAL Tools Panel (Recommended)

1. Open CAAL web interface
2. Click Tools panel (wrench icon)
3. Search for "espn-get-nhl-schedule"
4. Click Install and follow prompts

### Via Command Line

```bash
curl -s https://raw.githubusercontent.com/CoreWorxLab/caal-tools/main/scripts/install.sh | bash -s espn-get-nhl-schedule
```

## Usage

Get upcoming games for an NHL team. Required parameter: team (e.g., 'Canucks', 'Maple Leafs', 'TOR'). Examples: 'when do the Canucks play next', 'Leafs schedule', 'next Oilers game'.
